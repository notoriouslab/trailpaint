教會 × TrailPaint

把聖經故事與宣教史變成可分享的互動地圖。年代滑桿讓會眾拖一下，2000 年三段絲路足跡層層疊疊出現在同一張地圖上。

三個用例

主日學：四福音書地圖

馬太、馬可、路加、約翰各自記述的耶穌足跡——加利利傳道、變像山與耶路撒冷受難週。33 個景點橫跨 AD 27-30。

週報：保羅宣教三次旅行

從敘利亞安提阿出發，三度橫渡地中海，最後押解羅馬。沿使徒行傳 13-28 章足跡。約 34 個景點橫跨 AD 46-62。

個人靈修：絲路 2000 年

張騫鑿空 BC 138、玄奘西行 AD 629、馬可波羅東來 AD 1271——同一條中亞走廊，1400 年三段足跡層層疊疊。拖年代滑桿可看到不同時代逐一聚焦。

怎麼嵌進你的網站 / 週報 / 主日學投影片

每個合輯都可以一鍵複製 embed code 貼進你的場景：

  1. 打開合輯：四福音書地圖 · 保羅三次 · 絲路 2000 年
  2. 右下播放控制列點 📋 按鈕複製 iframe code
  3. 貼進 WordPress / Wix / 教會官網 / Google Slides（用 iframe 嵌入功能）

嵌入碼預設 100% 寬 / 500px 高。可改 width / height 適配週報排版。

還有什麼可以做

  • 產生電子明信片：拖年代滑桿到任一刻度，點 📮 把當下地圖 + 年代印章 + 經文出處下載成 1080×1080 IG 方圖
  • 切換歷史 / 聖經刻度：滑桿左側 🗺️ / 📖 兩組刻度，聖經組含出埃及 BC 1446 / 大衛 BC 1000 / 巴比倫 BC 586 / 耶穌 AD 30 / 保羅 AD 50 / 啟示錄 AD 95
  • 創建你自己的故事地圖：上傳照片 EXIF GPS、KML、GPX、或手動標點，做你的教會宣教史 / 短宣回顧 / 聖地考察記錄
