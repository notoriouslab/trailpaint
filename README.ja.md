## これは何？

TrailPaint 路小繪は手描き風のトレイルマップ作成ツールです。オンライン地図を使うか、スクリーンショットをアップロードして、数分で美しい共有用イラストマップを作成できます。

PWA 対応でスマホのホーム画面にインストール可能。中文 / English / 日本語を自動検出。

![trailpaint/examples/trailpaint-map.jpg](./examples/trailpaint-map.jpg)

## 使い方

```
1. 場所を検索する  または  スクリーンショットを地図にドラッグ
2.「スポット追加」をクリックしてマーカーを配置、名前と写真を入力
3.「ルート描画」でパスをなぞり、「ルート完成」をクリック
4.「書き出し」でプレビューを開き、比率・枠・スタイルを選んで PNG をダウンロード
```

初回起動時に 3 ステップのチュートリアル。ドロップダウンから台湾の有名トレイル 6 コースをサンプルとして読み込めます。

### AI でルートを作成

「インポート」→「🤖 AI でルート JSON を作成」→ プロンプトテンプレートをコピーして ChatGPT や Claude に旅程を説明すると、インポート可能な JSON ファイルが生成されます。

## 機能

### マップ

- 🗺️ **オンラインマップ** — OpenStreetMap、5 種のスタイル切替（標準 / 衛星 / 等高線 / ダーク / 多言語ベクター）
- 📷 **スクリーンショットベースマップ** — 画像をドラッグして背景マップに
- 🔍 **場所検索** — どこでも検索、地図がフライ
- ◎ **現在地** — ワンタップで位置情報取得

### マーカー

- 🖊️ **ルート描画** — 手描き風点線 + 方向矢印、5 色自動ループ
- 📍 **スポットカード** — 紙テクスチャの手描き風、写真・26 種アイコン・ドラッグ配置
- ✏️ **ルート編集** — ノードをドラッグして調整、ダブルクリックで削除、色変更
- 🔎 **ズーム連動カード** — カードが地図のズームに合わせてスケール
- 👆 **タッチドラッグ** — モバイルで指でカードをドラッグ

### データ

- 📥 **GPX インポート** — トラック + ウェイポイント解析、大量ポイント自動簡略化
- ⛰️ **標高プロファイル** — 距離 / 推定時間 / 累計上昇 & 下降
- 🔭 **標高クエリ** — 手動ルートも標高取得可能（Open-Meteo API）
- 📏 **ルート自動命名** — 逆ジオコーディングで地名取得
- 💾 **保存 / 読込** — JSON 形式、完全復元可能

### 書き出し

- 📐 **書き出しプレビュー** — リアルタイムプレビュー、全オプション一画面
- 🎯 **構図調整** — プレビュー内でパン・ズームして構図を微調整
- 📐 **マルチ比率** — 1:1（IG）/ 9:16（Story）/ 4:3 / オリジナル
- 🖼️ **3 種フレーム** — クラシック二重枠 / 紙手描き / ミニマル細枠
- 🎨 **4 種スタイルフィルター** — オリジナル / 水彩 / スケッチ / ヴィンテージ
- 📊 **統計オーバーレイ** — 距離・時間・獲得標高を画像に印字
- 🔗 **共有リンク** — 共有用 URL を生成（短縮リンク対応）
- 🤖 **AI プロンプト** — スタイル化プロンプトをワンクリックコピー

### インポート

- 📥 **インポートウィザード** — 統合インポート：背景画像 / JSON / GPX
- 🤖 **AI チュートリアル** — プロンプトテンプレート内蔵、AI で JSON 生成
- 🌿 **サンプルトレイル** — 台湾の有名コース 6 つをワンクリック読込

### 体験

- ↩️ **元に戻す / やり直し** — Cmd+Z / Ctrl+Z
- 🌊 **手描き揺れ効果** — SVG フィルターエフェクト
- 📱 **PWA** — ホーム画面にインストール、オフラインキャッシュ
- 🌐 **3 言語対応** — 中文 / English / 日本語、自動検出
- ☰ **モバイルメニュー** — サイドバー非表示時のフローティングアクションメニュー

## オフライン版

完全オフラインが必要な場合は [`trailpaint.html`](trailpaint.html) をダウンロードしてブラウザで開いてください。

> ⚠️ オフライン版にはオンラインマップ、GPX インポート、標高機能はありません。iOS ではローカル HTML を開けません。

## 技術構成

- **フレームワーク**: Vite + React 19 + TypeScript
- **地図**: Leaflet + react-leaflet + protomaps-leaflet（CARTO + Protomaps ベクタータイル）
- **状態管理**: Zustand + zundo（temporal undo/redo）
- **PWA**: vite-plugin-pwa + Workbox
- **書き出し**: html-to-image + Canvas 後処理 + スタイルフィルター
- **i18n**: カスタム t() + ランタイムロケール検出
- **構造**: core/（Leaflet 非依存）+ map/（Leaflet 統合層）

## 開発

```bash

npm run dev        # 開発サーバー
npm run build      # ../app/ へビルド
```
